
# AI Emotion Detection

## Overview
A real-time emotion detection project built using Python, MediaPipe, OpenCV, and TensorFlow.

## Technologies Used
- Python
- TensorFlow
- OpenCV
- MediaPipe
- NumPy

## Features
- Real-time webcam input
- Face detection
- Emotion prediction
- Live display of detected emotion

## What I Learned
- Building an AI pipeline
- Image preprocessing
- Integrating TensorFlow with OpenCV

## Future Improvements
- Improve accuracy
